# 🦷 AlignTrack

Smart Orthodontic Department Monitoring App built using Flutter and Firebase.

---

## 🚀 Features

- 🔐 Firebase Authentication (Login system)
- 👨‍⚕️ Role-Based Dashboards
  - HOD Dashboard
  - Faculty Dashboard
  - PG Dashboard
- 💬 Real-time Chat / Discussion
- 📤 Image Upload (Firebase Storage)
- 🗂 Case Tracking System

---

## 🛠 Tech Stack

- Flutter (Frontend)
- Firebase Authentication
- Cloud Firestore
- Firebase Storage
